Delicious Cake is a both NeoForge and Fabric mod and the successor to Yummy Cakes.

If you don't know what Yummy Cakes is, this mod adds some delicious cakes!

The current cakes include:
- Glazed Cake
- Golden Apple Cake
- Cookie Cake
- Nether Cake
